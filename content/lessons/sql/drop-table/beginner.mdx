Avoid errors if the table doesn't exist:


If other tables reference yours:

```sql
-- This fails if other tables have foreign keys to customers:
DROP TABLE customers;
-- Error: Cannot drop, referenced by FK

-- Solution 1: Drop referencing tables first
DROP TABLE orders;
DROP TABLE customers;

-- Solution 2: Drop the foreign key first
ALTER TABLE orders DROP CONSTRAINT FK_orders_customers;
DROP TABLE customers;
```
