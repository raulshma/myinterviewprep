# CONCAT and LENGTH: String Building Blocks

Combine strings and measure their length!

## ðŸ“ CONCAT Syntax by Database

```sql
-- Standard CONCAT
SELECT CONCAT(first, second, third);

-- SQL Server + operator
SELECT first_name + ' ' + last_name;

-- PostgreSQL || operator
SELECT first_name || ' ' || last_name;

-- MySQL CONCAT_WS (with separator)
SELECT CONCAT_WS(', ', city, state, country);  -- 'NYC, NY, USA'
```

## ðŸ“ LENGTH: Measuring Strings

```sql

```

## ðŸŽ¯ Practical Examples

```sql
-- Full name with title
SELECT CONCAT(title, ' ', first_name, ' ', last_name) AS formal_name
FROM employees;

-- Check string length
SELECT name FROM products WHERE LENGTH(name) > 50;

-- Username validation
SELECT username,
    CASE
        WHEN LENGTH(username) < 3 THEN 'Too short'
        WHEN LENGTH(username) > 20 THEN 'Too long'
        ELSE 'Valid'
    END AS status
FROM users;
```

## âš¡ NULL Handling

```sql
-- CONCAT with NULL
SELECT CONCAT('Hello', NULL, 'World');  -- 'HelloWorld' (NULL ignored in most DBs)

-- + operator with NULL (SQL Server)
SELECT 'Hello' + NULL + 'World';  -- NULL! (entire result is NULL)

-- Safe concatenation
SELECT CONCAT(COALESCE(first, ''), ' ', COALESCE(last, ''));
```
