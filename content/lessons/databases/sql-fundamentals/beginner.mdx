# SQL Fundamentals

Welcome to the world of databases! SQL (Structured Query Language) is the universal language for talking to databases. Let's learn how to speak it!

## üó£Ô∏è What is SQL?

Imagine you have a giant filing cabinet with thousands of folders. SQL is like having a super-smart assistant who can:

- **Find** any document you describe
- **Add** new documents
- **Update** existing documents
- **Remove** documents you don't need

> **Think of it this way:** SQL is like ordering at a restaurant. You tell the database what you want, and it brings it to you!

## The Four Magic Words: CRUD

Every database operation falls into four categories, known as **CRUD**:

| Operation | SQL Command | What It Does | Restaurant Analogy |
|:----------|:------------|:-------------|:-------------------|
| **C**reate | INSERT | Adds new data | Ordering new food |
| **R**ead | SELECT | Retrieves data | Looking at the menu |
| **U**pdate | UPDATE | Changes existing data | Modifying your order |
| **D**elete | DELETE | Removes data | Canceling an order |

## üîç WHERE - Filtering Your Data

The WHERE clause is like a filter - it lets you specify exactly which rows you want to work with.

```sql
-- Find users over 25
SELECT * FROM Users WHERE Age > 25;

-- Find users named 'Bob'
SELECT * FROM Users WHERE Name = 'Bob';

-- Combine conditions
SELECT * FROM Users WHERE Age > 25 AND Active = 1;
```
